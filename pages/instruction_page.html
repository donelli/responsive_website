Maneiras de instalar

Há diversas maneiras de instalar o Foundation em um projeto:

  • Utilizando Package Managers:
    npm install foundation-sites
    yarn add foundation-sites
    php composer.phar require foundation/foundation

  • Utilizando o CLI do próprio Fountation:
    npm install --global foundation-cli
    foundation new

  • Utilizar atraves de um CDN:
    <!-- Compressed CSS -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.6.3/dist/css/foundation.min.css" integrity="sha256-ogmFxjqiTMnZhxCqVmcqTvjfe1Y/ec4WaRj/aQPvn+I=" crossorigin="anonymous">

    <!-- Compressed JavaScript -->
    <script src="https://cdn.jsdelivr.net/npm/foundation-sites@6.6.3/dist/js/foundation.min.js" integrity="sha256-pRF3zifJRA9jXGv++b06qwtSqX1byFQOLjqa2PTEb2o=" crossorigin="anonymous"></script>

  • Há também como baixar um projeto base com o básico do framework.

  • O Fountation, assim como o Bootstrap, possui um maneira de customizar o framework, detalhes podem ser consultados aqui.

  • Pode ser utilizado em conjunto com diversos outros frameworks: Consulte aqui para mais detalhes

Nosso relato sobre a instalação em nosso projeto

Primeiramente baixamos o projeto base que o Foundation disponibiliza. Porém a build deste projeto é somente com algumas das funcionalidades que o framework disponibiliza. No nosso caso, queriamos todas para podermos testa-las. Então tivemos que ir atrás de outra maneira. Tentamos utilizar os links do CDN, porém a versão lá, era a mesma no projeto base disponibilizado. No final, tivemos que criar uma versão customizada com todas as funcionalidades.

Ao longo de várias horas tentando fazer tudo funcionar, continuamos com alguns erros no console sem saber o por que. Por sem o segundo "maior" framework, pensamos que seria mais fácil.
